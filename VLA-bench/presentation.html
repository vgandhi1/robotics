What Each Layer Buys

Optimizations applied in sequence

Phase 2 · WebDataset streaming

Removes CPU decode bottleneck — GPU utilization 40% → 87%.

Phase 3a · FlashAttention-2

Fused attention, no N² materialization — peak VRAM -26%.

Phase 3b · FSDP full shard

Enables 2× batch size; 88% multi-GPU scaling efficiency.

Phase 3c · Activation checkpointing

Extra -15% VRAM, trades ~20% compute overhead.

03
Why It Matters

Compresses the failure → patch → redeploy loop

On 2×RTX 3090 (RunPod $2.50/hr), a 500K-frame run drops from ~9.9 hrs ($24.75) to ~3.5 hrs ($8.75). In a sim-to-real pipeline this cuts the real-floor iteration cycle from ~2 days to ~6 hours — same-day iteration on new failure modes.
